### Opgave 8:

Arbejde videre med dit binære søgetræ fra sidste lektion.

Implementer følgende metoder:

```java
E removeMin(); // Fjerner det mindste element i træet og returnerer 
```

```java
E removeMax(); // Fjerner det største element i træet og returnerer 
```

```java
int greaterThanCount(E element); // Returnerer antallet af elementer i træet, der er større end det givne element
```

```java
List<E> greaterThan(E element); // Returnerer en liste over alle elementer i træet, der er større end det givne element
```
